Section 3: Lists

Lists is a collection of items. This can range from having the names of your family members to numbers to items that don't relate to each other at all.

To create a list, all you have to do is set a variable equal to [] (empty brackets). You can then add whatever items you would like inside the brackets such as numbers = [9, 1, 2, 3].

A cool thing about list is that you can individually access each element inside the brackets. However, be aware that the starting position is 0 and not 1. Using the previous example, to access the first element, type numbers[0] to get an output of 9.

Section 4: If Conditionals

If conditionals is used when you want something to happen if a certain condition has been met. To create an If conditional, type the following:

if (conditional statment here) {
output statement here;
}

A conditional statment may need a conditional operators. Some simple conditional operators includes:

Equals: == (two equal signs)
Less than: <
Less than or equal: <=
Greater than: >
Greater than or equal: >=

In practice, it should look something like this:

if (7 < 9):
print("7 is less than 9");
