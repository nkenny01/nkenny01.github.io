Print and Variables

Section 1: Print

The first lesson to learn about any programming language is to learn the syntax on how to print your desired output. In Python, the syntax is very simple. To print, all you have to type is print() with your desired output inside the parentheses. For example, print(9) will print out output 9 and print("Hello World") will print out Hello World. Section 2: Variables

Variables are used to store information in a placeholder. To use variables, you must declare a name for the variable then set it equal to something. In practice, it would look something like this:

name = "John Doe";
print(name);

When printing out the variable name, it will print John Doe. For more information, click the links below or the image to learn variables and type your own variables in a code editor in W3Schools.
